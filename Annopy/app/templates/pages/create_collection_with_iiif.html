Création d'une nouvelle collection avec IIIF

Pour créer une nouvelle collection, remplissez le formulaire et cliquer sur "créer".

Pour créer une nouvelle collection à l'aide de IIIF, il vous suffit d'indiquer le lien du manifest IIIF JSON que vous souhaitez récupéré, ainsi qu'un intervalle d'images à récupérer. Par exemple, si vous souhaitez récupérer les images 3 à 50 du manifest IIIF, indiquez 3 et 50. Nous nous chargerons ensuite de récupérer les URL de chaque image et de les associer à la collection que vous avez créé.

Exemple de manifest IIIF JSON valides :

  • https://ids.si.edu/ids/manifest/NMAAHC-2012_164_125_001
  • https://gallica.bnf.fr/iiif/ark:/12148/bpt6k9923506/manifest.json
